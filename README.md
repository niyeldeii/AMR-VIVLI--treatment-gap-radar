# Treatment Gap Radar

Vivli AMR Data Challenge submission (AMR ID 00013367). Builds a framework that overlays a
**Resistance Need Index** (from Vivli surveillance data) against an **R&D Attention Index**
(from the Global AMR R&D Hub export) to surface pathogen–drug "treatment gaps" — where
resistance pressure is high but R&D investment is low.

See **[PLAN.md](PLAN.md)** for the full implementation plan.

## ⚠️ Data is NOT in this repository

The Vivli surveillance datasets and the `Projects.xlsx` R&D Hub export are **restricted under the
Vivli Data Use Agreement** and are deliberately excluded from git (see `.gitignore`). They live
locally alongside this folder and are referenced by relative path. Do **not** commit or upload
them.
